# ValxGuide 🎮

**Aplicativo de Guia** do Jogo Valorant, contendo várias informações de Agentes, Ranks, Sprays, Cards de jogadores, Mapas e Chaveiros.

## Tecnologias usadas 🚀

- Android | Kotlin ✔︎
- Koin ✔︎
- MVVM + Clean Architecture ✔︎
- Retrofit ✔︎
- Coroutines ✔︎
